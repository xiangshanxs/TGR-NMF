# Deciphering progressive lesion areas in breast cancer spatial transcriptomics via TGR-NMF
The Demo. m file is the main file.
The EuDist2. m file is used to calculate the Euclidean distance between spots.
ConstructW1. m, ConstructW1. m, and ConstructW1. m files are used to calculate adjacency matrices W1, W2, and W3, respectively.
The GNMF_Multi.m file is used to calculate the low-dimensional representation V of the gene expression count matrix.
The Unitization.m file is used to preprocess the gene expression count matrix and position coordinate matrix.
The Litekmeans.com file is used for clustering.
The Cal-ARI. m file is used to calculate the adjusted rand index.
GSCAC_ClusteringMeasure. m and Contingency.m files are used to calculate the gold standard comparison accuracy.
